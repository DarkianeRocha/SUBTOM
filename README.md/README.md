## 📌 Sobre o projeto

Este projeto foi desenvolvido inicialmente como atividade acadêmica e poderá ser evoluído futurament como projeto de estudo em Front-end.
A ideia era simples: digitar um código HEX e mostrar informações reais daquela cor na tela.
Pra isso, eu usei uma API que retorna os dados da cor e mostro tudo no site.

---

## 🌐 API publica usadam

Usei a **The Color API**  
https://www.thecolorapi.com/

Ela basicamente pega o HEX e devolve informações como nome da cor, RGB, HSL e CMYK.

---

## ⚙️ O que foi usado

- HTML
- CSS (Tailwind)
- JavaScript
- Fetch API
- The Color API

---

## 🎨 O que o projeto faz

- Busca uma cor pelo código HEX
- Mostra o nome da cor
- Mostra RGB, HSL e CMYK
- Mostra a cor na tela
- Gera uma paleta de tons
- Valida o que o usuário digita
